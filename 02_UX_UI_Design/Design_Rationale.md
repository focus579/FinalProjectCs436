# เหตุผลในการออกแบบ (Design Rationale)

### 1. หน้าตาโดยรวม (Visual Design)
* **สไตล์ iPhone:** ผมออกแบบปุ่มให้เป็นวงกลมและวางเรียงกันแบบ Grid เหมือนเครื่องคิดเลขใน iPhone เพราะเป็นรูปแบบที่คนคุ้นเคยที่สุด ใช้งานเป็นได้ทันทีโดยไม่ต้องงมหาปุ่ม
* **พื้นหลัง (Background):** ใช้การไล่สี (Gradient) และใส่ Animation ให้สีค่อยๆ ขยับเปลี่ยนไปเรื่อยๆ เพื่อให้เว็บดูมีชีวิตชีวา ดูโปรและทันสมัยกว่าเว็บทั่วไปที่เป็นสีพื้นนิ่งๆ

### 2. การจัดวาง (Layout)
* **Dual Display:** แยกส่วนแสดง "โจทย์" (ตัวเล็กสีเทา) กับ "ผลลัพธ์" (ตัวใหญ่สีขาว) ออกจากกัน เพื่อลดความสับสนเวลาบวกเลขหลายชั้น
* **การจัดกลุ่มปุ่ม:**
    * **สีส้ม:** ไว้ขวาสุดสำหรับเครื่องหมายคำนวณ (Action Buttons)
    * **สีเทาอ่อน:** ไว้บนสุดสำหรับคำสั่งพิเศษ (AC, ลบ)
    * **สีเทาเข้ม:** สำหรับตัวเลข

### 3. Feedback
* ใส่ Effect เวลาเอาเมาส์ไปกดปุ่ม ปุ่มจะยุบลงนิดนึงและสว่างขึ้น ให้ความรู้สึกเหมือนกดปุ่มจริงๆ
