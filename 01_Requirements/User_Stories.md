 User Stories
1. ในฐานะผู้ใช้ทั่วไป ฉันต้องการกดตัวเลขและเครื่องหมายบวก เพื่อให้ได้ผลลัพธ์การบวก
2. ในฐานะผู้ใช้ทั่วไป ฉันต้องการกดเครื่องหมายลบ เพื่อให้ลบตัวเลขได้
3. ในฐานะผู้ใช้ทั่วไป ฉันต้องการกดเครื่องหมายคูณ เพื่อให้คูณตัวเลขได้
4. ในฐานะผู้ใช้ทั่วไป ฉันต้องการกดเครื่องหมายหาร เพื่อให้หารตัวเลขและได้รับข้อความผิดพลาดเมื่อหารด้วยศูนย์
5. ในฐานะผู้ใช้ทั่วไป ฉันต้องการปุ่ม C เพื่อรีเซ็ตหน้าจอ
6. ในฐานะผู้ใช้ทั่วไป ฉันต้องการปุ่ม Backspace เพื่อแก้ไขตัวเลขที่พิมพ์ผิด
7. ในฐานะผู้ใช้ทั่วไป ฉันต้องการใส่จุดทศนิยม เพื่อคำนวณตัวเลขทศนิยมได้
8. ในฐานะผู้ใช้ทั่วไป ฉันต้องการเห็นสถานะของการคำนวณ (expression) เช่น "12 +" ก่อนที่จะแสดงผล เพื่อให้รู้ว่าระบบกำลังทำงานอย่างไร
9. ในฐานะผู้ทดสอบ ฉันต้องการ Test Cases อย่างน้อย 10 เคสเพื่อยืนยันฟังก์ชันหลัก
10. ในฐานะผู้ส่งงาน ฉันต้องการ README และคู่มือผู้ใช้ เพื่อให้ผู้ตรวจงานสามารถรันโปรเจกต์ได้ทันที
