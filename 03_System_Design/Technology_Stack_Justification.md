# เหตุผลที่เลือกใช้เครื่องมือ (Tech Stack)

### 1. HTML5
* ใช้สร้างโครงสร้างปุ่มและหน้าจอแสดงผล

### 2. CSS3 
* โปรเจคนี้เน้น CSS มากเป็นพิเศษ โดยเฉพาะเรื่อง **Animation** ที่ใช้ทำพื้นหลังขยับได้
* ใช้ **CSS Grid** และ **Flexbox** ในการจัดปุ่มให้ตรงกันเป๊ะๆ รองรับหน้าจอขนาดต่างๆ

### 3. JavaScript
* ใช้จัดการ Logic การคิดเลขทั้งหมด
* เขียนแบบแยกฟังก์ชันชัดเจน (เช่น ฟังก์ชันคำนวณ, ฟังก์ชันอัปเดตหน้าจอ, ฟังก์ชันล้างค่า) เพื่อให้แก้โค้ดง่ายในอนาคต
