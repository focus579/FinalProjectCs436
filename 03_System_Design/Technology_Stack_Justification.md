# เหตุผลที่เลือกใช้เครื่องมือ (Tech Stack)

### 1. HTML5
* เป็นพื้นฐานในการสร้างโครงร่างของเว็บ ปุ่มต่างๆ และหน้าจอแสดงผล

### 2. CSS3
* ใช้ตกแต่งหน้าตาให้สวยงาม
* เลือกใช้ **CSS Grid** เพราะจัดระเบียบปุ่มที่เป็นตารางได้ง่ายที่สุด ไม่ต้องมานั่งกะระยะเอง

### 3. JavaScript (แบบพื้นฐาน)
* ใช้เขียนคำสั่งคำนวณเลข (+ - * /)
* ใช้รับค่าจากการกดปุ่ม (Click Event) แล้วเอาไปแสดงบนหน้าจอ
* ที่ไม่ใช้ Framework (เช่น React) เพราะโปรเจคนี้ไม่ซับซ้อนมาก เขียนด้วย JavaScript ธรรมดาจะทำงานได้เร็วกว่าและไฟล์เล็กกว่า
