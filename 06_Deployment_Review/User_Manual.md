# คู่มือการใช้งาน

### ส่วนประกอบหน้าจอ
1. **จอบน (ตัวเล็ก):** แสดงเลขที่เรากำลังคำนวณค้างไว้
2. **จอล่าง (ตัวใหญ่):** แสดงผลลัพธ์ปัจจุบัน

### วิธีใช้งาน
1. กดตัวเลขที่ต้องการ
2. กดเครื่องหมาย (+ - × ÷) -> เลขจะเด้งขึ้นไปรอข้างบน
3. กดตัวเลขตัวถัดไป
4. กดปุ่มเท่ากับ (=) เพื่อสรุปยอด

### ปุ่มลัด
* **⌫ (Backspace):** ลบตัวเลขตัวล่าสุด (กรณีมือลั่นกดผิด)
* **AC (All Clear):** ล้างทุกอย่าง เริ่มใหม่หมด
